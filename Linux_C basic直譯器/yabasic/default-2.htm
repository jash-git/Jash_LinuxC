Yabasic in a Nutshell

Yabasic implements the most common and simple elements of the basic langugage; It comes with goto/gosub, with various loops and with user defined subroutines. Yabasic does monochrome line grafics, printing comes with no extra effort. Yabasic runs under Unix and Windows; it is small (around 200KB) and free (i.e. subject to the GNU copyleft).

Binaries for Windows

Yabasic runs under Windows 95, 98 and NT; this is how to install:

  • Click here (200 kB) to download the self-extracting archive yabbin.exe. Put this archive in a temporary directory. This directory can be removed after successfull installation.
  • double-click on yabbin.exe
  • double-click on setup.exe
  • follow the instructions

The setup-program does the following:

  • Make an entry in the start-menu
  • Registers the extension ".yab"
  • Make an entry in the Root-Menu
  • Put default-settings into the registry
  • Register it's uninstall-program
  • Write a complete installation-log in Windows/Temp/yabasic.log

To remove yabasic, go to the control-panel and click on the software-icon: You will find an entry for deleting yabasic.

If anything goes wrong please give me a mail: mail@yabasic.de

Binaries for Linux

This is how to get the redhat package of yabasic for linux-i386:

  • Click here (100 KB) to download the redhat package yabasic.rpm.
  • Type rpm -i --force --nodeps yabasic.rpm ; the --force switch causes rpm to superseed an existing (older) version of yabasic. --nodeps is necessary, because my version of rpm (2.5.5) introduces some silly dependencies; hope I will be able to fix this soon. You need to be root to do this.
  • Read the documentation under /usr/doc/yabasic
  • or try the demos under /usr/share/yabasic
  • to remove yabasic type rpm -e --allmatches yabasic ; again this can only be done as root.

This binary distribution can be installed under Linux with an Intel CPU and rpm (the redhat package manager) only. If your system doesn't match these requirements, or if you want to see the sources you may download them from this page.

Once you have grabbed yabasic, you may download the sokoban game written by Hermang H. Mansilla; it's fun to play and shows what can be done with yabasic ! Just go to www.geocities.com/SunsetStrip/Palms/1624/yabasic/

If you have any problems, please drop me a mail: mail@yabasic.de.

Compile sources for Unix

This is how to compile yabasic in your home-directory:

  • Click here (250 KB) to download its tar-file.
  • Some browsers may download the file as yab_tar.tar (instead of the correct name yab.tar.Z) in this case you should rename it: mv yab_tar.tar yab.tar.Z
  • Uncompress the tar-file: uncompress yab.tar.Z
  • tar xf yab.tar, this unpacks the tar-file, giving you a directory named yabasic-2.xx (where 2.xx is the current version of yabasic)
  • cd yabasic-2.xx
  • runme, this starts a shell script which builts the executable program.
  • yabasic demo.yab, this executes a tiny yabasic-program, showing some grafics.
  • If you have root privileges, you may try make install to install yabasic in your system.

Yabasic needs the X and ncurses libraries to compile. Yabasic compiles under HPUX and linux, but chances are high, that users of other mainstream Unixes will have no problems. If anything goes wrong please give me a mail: mail@yabasic.de, including:

  • the sort of system you are working on and
  • the file runme.log which is written during execution of the script runme

I will try to find and fix the problem.
