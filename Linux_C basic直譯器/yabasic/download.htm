Upgrading from an earlier Version

There is no special procedure necessary to upgrade from a previous Version of yabasic.

So, If you already have yabasic installed, or if you do a fresh install, just follow the instructions given below.

Binaries for Windows

Yabasic runs under any version of Windows; this is how to install:

  • Download yabasic-2.763.bin.exe (260 kB); save it on your harddisk.
  • double-click on the file
  • follow the instructions

The setup-program does the following:

  • Make an entry in the start-menu
  • Register the extension ".yab"
  • Make an entry in the Root-Menu (this is the menu, which appears if you click the right Mousebutton on an empty part of your Desktop)
  • Put default-settings into the registry
  • Register its uninstall-program
  • Write a complete installation-log in Windows/Temp/yabasic.log

To remove yabasic, go to the control-panel and click on the software-icon: You will find an entry for deleting yabasic.

If anything goes wrong please give me a mail: mail@yabasic.de 

Binaries for Linux

This is how to get the redhat package of yabasic for linux-i586:

  • Click here to download yabasic-2.763-1.i586.rpm (220 KB), the redhat package of yabasic.
  • Type rpm -i --force --nodeps FILE (where FILE is the full name of the file you have just downloaded); the --force switch causes rpm to superseed an existing (older) version of yabasic. --nodeps is necessary, because my version of rpm (2.5.5) introduces some silly dependencies; hope I will be able to fix this soon. You need to be root to do this.
  • Read the documentation under /usr/doc/yabasic
  • or try the demos under /usr/share/yabasic
  • to remove yabasic type rpm -e --allmatches yabasic ; again this can only be done as root.

This binary distribution can be installed under Linux with an Intel CPU and rpm (the redhat package manager) only. If your system doesn't match these requirements, or if you want to see the sources you may download them from this page.

If you have any problems, please drop me a mail: mail@yabasic.de.

Compile the sources for Unix

Compiling yabasic requires a decent C-Compiler, the ncurses-library and the X-Window libraries. To compile yabasic in your home-directory proceed like this:

  • Click here to download yabasic-2.763.tar.gz (490 KB) the tar-file of yabasic.
  • Uncompress the tar-file: gunzip FILE (where FILE is the full name of the file you have just downloaded).
  • tar xf yabasic-x.xxx.tar, this unpacks the tar-file, giving you a directory named yabasic-x.xxx (where x.xxx is the current version of yabasic)
  • cd yabasic-x.xxx
  • runme, this starts a shell script which builts the executable program. (Alternatively you may type: ./configure && make && make check)
  • If you have root privileges, you may try make install to install yabasic in your system.

Yabasic needs the X and ncurses libraries to compile. Yabasic compiles under HPUX and linux, but chances are high, that users of other mainstream Unixes will have no problems. If anything goes wrong please give me a mail: mail@yabasic.de, including:

  • the sort of system you are working on and
  • the file runme.log which is written during execution of the script runme

I will try to find and fix the problem.

Sources for Windows

If you'd like to get the Windows specific files (including the sources of the installation program), you may download yabasic-2.763.src.exe (60 KB). This file does not contain the sources for yabasic itself (only the project files), so in addition you will have to download the sources for unix (there is no difference between Unix and Windows sources) and import them into the project. To compile yabasic you need MS Visual C++, Version 6.0.

If anything goes wrong please give me a mail: mail@yabasic.de
