The old documentation of yabasic

The old documentation of yabasic has been replaced in August 2003 by a complete rewrite, the new documentation.

However, the old documentation still has its merits:

  • For one it takes a more holistic approach; i.e. in most cases a single example explains many different features. So with the old documentation more things can be learned with a short read (albeit not in every detail).
  • Second, there is a german translation of the old documentation.

However, keep in mind, that the old documentation has not been updated from August 2003 on, i.e. it only documents the version 2.72 of yabasic. New features are only described within the new documentation.
