一、光碟內容

    本光碟是《Linux環境下C編程指南（第2版）》一書的配書光碟，內容包括了書中所有示例的根源程式代碼。 

    本光碟根目錄下包括檔夾source code和一個readme文字檔案，其內容和含義說明如下:

    (1) source code目錄中包含了本書所有章節示例的源代碼。各個章節內容分別置於以chapter_x命名的檔夾，各個檔夾包含對應章節出現的示例代碼，示例代碼建議用寫字板打開。

    (2) readme文字檔案是本書光碟的使用說明。 


二、運行環境
    1. 硬體要求
     CPU：1.6G以上，記憶體：1G以上，硬碟：32G以上。 
 
    2. 安裝Linux作業系統及開發包
     本書的示例程式，已經在基於Linux的Fedora 10平臺上成功的測試。
     Fedora 10 dvd版安裝: 按照提示步驟，磁片分區結束後，會提示選擇是否安裝附帶的開發包以及工具，選擇“現在定制”，然後在列出
的項目中選中“選擇安裝automake 工具和gcc/g++編譯器”。然後開始安裝，安裝結束後即可運行本書中的程式，無需其他配置。
     Fedora 10自帶的gcc版本為4.3.2，make版本為3.81。
