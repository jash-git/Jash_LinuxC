Cgicc在Linux下的开发

站长原创，版权所有ITEEDU，2011-07-24

在Linux下边开发cgicc应用程序之前，需要以下几个步骤：

（1） 先安装apache2：sudo apt-get install apache2:；

（2） 启动apache2：sudo /etc/init.d/apache2 restart；

（3） 编译和安装完毕cgicc；

（4） 在eclipse中开发cgicc应用程序，首先需要进行环境变量设置：

     

在eclipse的工程项目中，进行设置：

（1）右击，在弹出的右键快捷菜单中选择“属性”，在打开的面板中，展开“C/C++Build”，如图1所示；

（2）并在展开项中选择“setting”，如图2所示；

（3）设置includes选项，来包含所引用的头文件所在的位置，如图3所示，在图3中，其中的/usr/include/cgicc，是cgicc所用到头文件所在的路径。

（4）设置设置libraries选项，来包含所用到的库文件所在的路径，如图4所示，在图4中，其中libraries(-L)中，/usr/lib是cgjcc所在的库路径的设置，同时在libraries(-l)中，cgicc是所用到的库，该库的原名成为libcgicc.so，但是在这里设置需要去掉头(lib)和尾(.so)。

（5）单击“确定”按钮，至此之后，在eclipse中的设置完毕，可以进行编写代码，编译和运行了…… 呵呵。
