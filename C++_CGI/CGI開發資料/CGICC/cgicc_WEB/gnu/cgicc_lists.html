Mailing Lists

GNU cgicc has three mailing lists: help-cgicc@gnu.org, info-cgicc@gnu.org, and bug-cgicc@gnu.org.

  • help-cgicc is the main discussion list and is used to discuss all aspects of cgicc, including development and porting.
  • info-cgicc is where announcements about cgicc are made.
  • bug-cgicc is a special list used for reporting bugs. For details on submitting a bug report, please see the Reporting Bugs section.

All of these lists are managed by mailman, and can be subscribed to, unsubscribed to, and configured via a web interface by following the links above. Web archives are also available.
