標題 : 在 linux 平台下開發 openGL 程式
作者 : 創意智多星
發表時間：8/21/2003 11:07:04 AM
Linux 平台可以大量減少研發(研究與發展)的費用，所以在 Linux 開發系統有許多優勢，openGL 的程式也一樣。

這裡有一個好上手的方式：
首先請到 http://www.openGL.org/ 取得規格書…等第一手資料。
http://www.opengl.org/developers/documentation/index.html
像是設計原則、圖形介面、GLUT 規格書，請先看熟這些文件。

再來到 http://www.mesa3d.org/ 安裝 MesaLib-5.0.1.tar.gz 與 MesaDemos-5.0.1.tar.gz

再來取得各個自學手冊，像是這個網址有一個自學手冊
http://www.eecs.tulane.edu/www/Terry/OpenGL/

編譯時下這樣的指令就可以了
gcc -o test test.c -lglut -lGL -lGLU -lm
(我的電腦上)
