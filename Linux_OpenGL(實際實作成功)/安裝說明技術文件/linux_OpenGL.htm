$gcc -o example example.cpp -lGL -lGLU -lglut -lm
这是编译时带opengl支持的程序写法.
-lGL:带基本库
-lGLU:带实用库
-lglut:带glut扩展库
-lm:带数学库.
头文件中加入的是:
#include <gl/gl.h>
#include <gl/glu.h>
#include <gl/glut.h>
#include <math.h>
建议楼主用qt编写程序,用kdevelop编译,这样要方便很多.
