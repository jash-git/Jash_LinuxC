TextType

设置字符的结构 
typedef struct TextType 
{
	std :: string	name;		// 字体的名称
	int				size;		// 字体的大小
	bool			bold;		// 是否使用粗体
	bool			italic;		// 是否使用斜体
	bool			underline;	// 是否使用下划线
	float			depth;		// z方向的深度
}TextType;

成员变量

name
字体的名称

size
字体的大小

bold
是否使用粗体

italic
是否使用斜体

underline
是否使用下划线

depth
z方向的深度
