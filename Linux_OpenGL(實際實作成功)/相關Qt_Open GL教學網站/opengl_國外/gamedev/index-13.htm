Desura Client Open Sourced!

Desura, the game distribution service that is similar to Steam, has released the source code of their client software under the GPL V3 License.

The plan behind the release of the code is to create a community around the service that will contribute bug fixes and features. If you fancy taking a look at this code you can find it over on GitHub.

Alternatively, if you just want to check out the Desura service you can download the client from the Desura website.
